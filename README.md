# EduBot 🎓 — Chatbot for College Inquiry

EduBot is an AI-powered virtual assistant for SMS College Varanasi.
It helps students, faculty, and visitors get instant answers about
courses, admissions, faculty details, placements, fees, and more.

## 🌐 Live Demo
👉 https://edubot-7jmy.onrender.com

## ✨ Features
- 💬 WhatsApp Bot via Twilio
- 🌐 Web Chat Interface
- 🎓 Student Section — UG & PG courses
- 👨‍🏫 Faculty Section — Department wise faculty list
- 🏫 Visitor Section — Admissions, Exams, Placements
- 📋 Miscellaneous — Fees, Scholarship, Hostel, Law College

## 🛠️ Built With
- Python
- Flask
- Twilio API
- HTML / CSS / JavaScript
